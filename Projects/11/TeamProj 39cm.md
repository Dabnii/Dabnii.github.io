## 👩‍👩‍👧‍👦 39cm 팀

## 📌 39cm Features

- 🟢 `동근 BE` - DB구축, 회원가입, 로그인, 찜하기, 마이페이지 유저정보
- 🟢 `상엽 BE` - DB구축, 상품리스트 조회, 장바구니, 결제
- 🔵 `동기 FE` - 베스트 상품 페이지 리스트, 마이페이지
- 🔵 `동섭 FE` - 장바구니, 결제페이지, 회원가입, 로그인, 메인
- 🔵 `다빈 FE` - 제품 상세 페이지, 네비게이션, 네비게이션 검색 바, 메인 페이지
