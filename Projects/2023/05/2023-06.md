   1. linear list of children
   2. requires a **`List<Widget>`**
   - code
     ```dart
     CustomScrollView(
       slivers: [
         SliverList(
           delegate: SliverChildListDelegate(
             [
               Text('Item 1'),
               Text('Item 2'),
               Text('Item 3'),
             ],
           ),
         ),
       ],
     )
     ```
2. `SliverChildListDelegate`

   1. a grid of slivers with a fixed `number of items in each row`
   2. requires the **`crossAxisCount`**

   - code

     ```dart
     //SliverGridDelegateWithFixedCrossAxisCount

     CustomScrollView(
       slivers: [
         SliverGrid(
           delegate: SliverChildBuilderDelegate(
             (BuildContext context, int index) {
               return Container(
                 color: Colors.blue,
                 child: Text('Item $index'),
               );
             },
             childCount: 10,
           ),
           gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
             crossAxisCount: 2,
             mainAxisSpacing: 10,
             crossAxisSpacing: 10,
           ),
         ),
       ],
     )
     ```

3. `SliverFixedExtentList`
   1. slivers with children of the same fixed extent (height or width)
   2. requires the **`itemExtent`**
   3. `offset 0 and without gaps`
   - code
     ```dart
     CustomScrollView(
       slivers: [
         SliverFixedExtentList(
           delegate: SliverChildBuilderDelegate(
             (BuildContext context, int index) {
               return Container(
                 height: 50,
                 color: Colors.blue,
                 child: Text('Item $index'),
               );
             },
             childCount: 10,
           ),
           itemExtent: 50,
         ),
       ],
     )
     ```
